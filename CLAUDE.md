# english-iso プロジェクト

## 概要

英語学習をテーマにした静的HTMLサイト。文法用語を使わず、フレーズの音声再生・発音練習・ゲーミフィケーション（XP・レベル・連続学習）を組み合わせた学習体験を提供する。

## ファイル構成

| ファイル | 内容 |
|---|---|
| `page0.html` | 第0部 ― はじめに・サバイバル英語 |
| `page1.html` | 第1部 ― 音と文字（発音記号・フォニックス） |
| `page2.html` | 第2部 ― 英語の骨組み（語順・文の型） |
| `page3.html` | 第3部 ― 広げる（フル密度版） |
| `pending/` | 今後追加予定のファイルを格納するディレクトリ（.gitignore済み） |

## 学習フロー

利用者は page0 から順番に進める設計。各ページ下部の「Next」セクションをクリックすると次のページへ遷移する。
